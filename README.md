# EverythingForLinux
Everything For Linux.用来辅助在linux下查找文件，分为模糊查找个精确查找俩种查找方式。

## 成功第一步，先建个仓库！

## 实现
UI:Qt和DTK(Deepin Tool Kit)做界面。
搜索算法：没有查找算法，直接调用find和locate，所以依赖find和locate工具。

## 功能
- 精确查找
- 模糊查找
- 托盘隐藏、退出
- 右键打开文件位置、复制文件、复制文件路径
